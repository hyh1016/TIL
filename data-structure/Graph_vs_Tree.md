# Graph vs Tree

## Tree vs Graph

### 공통점

- Node와 Edge로 구성됨
    - Node: 하나의 지점. 보통 데이터를 가짐
    - Edge: Node를 잇는 선

### 차이점

- Tree는 Graph 중 특정 조건을 만족하는 일부 그래프를 칭하는 용어
- Tree는 Cycle이 존재하지 않음
- Tree는 한 Node에서 다른 한 Node로 이동하는 경로가 단 한 가지(유일)
- Tree는 계층 구조(부모-자식)를 가짐
- Graph는 단방향 또는 양방향이 존재 (즉, 방향성이 있을 수도 있고 없을 수도 있음)
